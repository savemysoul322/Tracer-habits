# Трекер привычек — оконная версия (GUI)

Кроссплатформенное настольное приложение для ежедневного учёта привычек.
Позволяет добавлять привычки, отмечать их выполнение за любой день, видеть
серии выполнения (streak), оценку вероятности срыва и тепловую карту активности
в двух форматах. Данные сохраняются между запусками.

Файл программы: **`tracker-habits-final.py`**

---

## Возможности

- Добавление и удаление привычек прямо в окне.
- Отметка выполнения за **любой день** (переключение даты кнопками ◀ ▶ и кнопкой «Сегодня»).
- Сохранение данных между запусками (файл `~/.habit_tracker/data.json`).
- Алгоритмическое ядро: текущая и максимальная серия выполнения (streak),
  оценка вероятности срыва по взвешенному скользящему среднему.
- Тепловая карта активности в **двух форматах**: линейном (30 дней) и недельном
  (календарь). На обеих картах показаны числа дней месяца и подписи месяцев.
- Таблица недельной и месячной статистики.
- Экспорт отчёта в CSV.

## Требования

- **Python 3.8** или новее.
- Модуль **tkinter** (графический интерфейс).
  - Windows и macOS: tkinter входит в стандартную установку Python — ничего ставить не нужно.
  - Linux: на части дистрибутивов tkinter ставится отдельным пакетом, например:
    - Debian/Ubuntu: `sudo apt install python3-tk`
    - Fedora: `sudo dnf install python3-tkinter`
    - Arch: `sudo pacman -S tk`
- Сторонние библиотеки **не требуются** — используется только стандартная библиотека Python.

Проверить, что tkinter доступен:

```bash
python -c "import tkinter; print('tkinter OK')"
```

## Кроссплатформенность

Приложение работает на **Windows, macOS и Linux** без изменений кода:

- интерфейс построен на `tkinter` (одинаков на всех ОС);
- пути к файлам формируются через `pathlib` и `Path.home()`, поэтому

## Как пользоваться

### Отметка привычек за день
Слева в блоке **«Отметки за день»** перечислены привычки с галочками.
Галочка ставит/снимает отметку выполнения за **выбранный день**.
Рядом с каждой привычкой показан прогноз `P=...%` — вероятность срыва.

### Навигация по дням
- **◀ / ▶** — перейти на предыдущий/следующий день (в пределах последних 30 дней).
- **Сегодня** — вернуться к текущей дате.
- Выбранный день подсвечивается оранжевой рамкой на тепловой карте.

### Добавление и удаление привычек
- Введите название в поле внизу левого блока и нажмите **«+ Добавить»** (или Enter).
- Кнопка **✕** справа от привычки удаляет её вместе со всей историей (с подтверждением).

### Тепловая карта активности
Переключатель над картой выбирает формат:
- **Линейная (30 дней)** — строки соответствуют привычкам, столбцы — дням.
  Сверху показаны подписи месяцев и числа дней. Нижняя строка «ИТОГО/день» —
  агрегированная активность за день.
- **Недельная (календарь)** — строки соответствуют дням недели (Пн…Вс),
  столбцы — неделям; число дня выводится внутри клетки, а её цвет (оттенки
  зелёного) показывает долю выполненных привычек за этот день.

### Демо-данные (синтетика)
В верхней панели:
- **Синтетика: неделя** — добавляет демонстрационные данные за 7 дней;
- **Синтетика: месяц** — за 30 дней.

Данные дозаполняются к текущим привычкам (с подтверждением). Для реальной работы
синтетика не нужна — достаточно добавить свои привычки и отмечать их каждый день.

### Экспорт отчёта
Кнопка **«Экспорт (CSV)»** сохраняет журнал выполнения за 30 дней в CSV-файл
(строки — даты, столбцы — привычки, значения 0/1).

## Где хранятся данные

Все привычки и отметки автоматически сохраняются в файл:

```
~/.habit_tracker/data.json
```

(где `~` — домашняя папка пользователя: например `C:\Users\Имя` в Windows,
`/home/имя` в Linux, `/Users/имя` в macOS).

Сохранение происходит после каждого изменения и при закрытии окна, а загрузка —
при следующем запуске. Чтобы начать с чистого листа, удалите этот файл.

Формат файла — JSON:

```json
{
  "habits": ["Зарядка", "Чтение"],
  "log": {
    "Зарядка": { "2026-06-16": true }
  }
}
```

## Алгоритмическое ядро

- **Текущая серия (streak)** — число подряд идущих выполненных дней, считая с
  последнего дня периода. Прерывается на первом невыполненном дне.
- **Максимальная серия** — самая длинная серия выполнения за весь период.
- **Скользящее среднее** — средняя доля выполнения за последние 7 дней.
- **Вероятность срыва** считается по формуле:

  ```
  P(срыв) = 1 − взвешенное скользящее среднее за 7 дней
  ```

  Более свежие дни имеют больший вес (линейные веса). Чем чаще пропуски в
  последние дни, тем выше вероятность срыва. Значения `P ≥ 0.5` помечаются как
  риск (красным цветом в таблице и у привычки).

Основные параметры заданы константами в начале файла:

| Константа    | Значение | Назначение                              |
|--------------|----------|-----------------------------------------|
| `WINDOW`     | 7        | окно скользящего среднего (дней)         |
| `PERIOD`     | 30       | период для карты и месячной статистики   |
| `DATA_FILE`  | `~/.habit_tracker/data.json` | путь к файлу данных  |
